# 블록체인
- 블록을 변경, 삭제할 수 없는 거대한 데이터베이스 (Append Only)

# 블록체인 장점
- 블록을 변경할 수 없다.
  - 블록 생성 시 지문이 생성 되는데 자신의 블록 전에 있는 지문을 참고해서 생성 된다.
    (이전 블록 지문 + 현재 블록 지문)
  - 지문 변경 시 전체 그룹이 알게 되기 때문에 변경이 불가능

- ex) 비트코인 : 블록 안에 거래 내용이 담겨져 있음 (A가 B에게 돈을 줌)

# 암호 화폐
- 정부에 따라서 화폐 가치가 정해지는데, 암호 화폐의 경우에는 가치가 동일
  (정부가 아닌 수학을 믿음)

# 화폐의 속성
- Scarcity (유한성) : 비트코인의 경우도 2천 1백만개
- Fungibility (교환 가능성)
- Divisibility (나눌 수 있음)
- Durability (견고함. 변질X)
- Transferability (이동이 편리)
- Stability (가치의 안정성) : 현재 비트코인은 가치가 요동치기 때문에 안정적이지는 않음

# 비트코인이 해결하는 이슈
- Decentralization (탈 중앙화) : 하나의 조직이 화폐를 컨트롤하지 않음
- Peer 2 Peer : 제 3자의 간섭이 없음(중개인) ex) 시청자 ---> youtube(중개) ---> 영상 창작자
- Pseudonymous (익명성 보장) : 완벽한 익명은 아니지만, 필명과 가명을 보장
  (익명성 보장은 이론적인 것. 실제로는 거래소에서 비트코인을 구매함.)
- Instant Transaction (즉시 거래 가능) : 은행 승인 X ---> 컴퓨터에서 컴퓨터로 바로 거래 가능
  (즉시 거래 가능은 이론적인 것. 실제로는 현재 네트워크가 방대하기 때문에 거래가 오래 걸림.)

위의 4가지 문제들은 '현금' 이라면 가능한 이야기 이지만, 인터넷에서는 '현금'이 없음.
그렇기 때문에 인터넷을 위한 돈인 '비트코인'이 탄생
